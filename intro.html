Introduction

The following report approaches a proposed visualization solution for large amounts of data regarding the evolution of metrics related to fuel prices.

This work has been done with the knowledge acquired during the semestral subject of Information Visualization and makes use of data provided by Prio displayed using graphs and plots developed with the use of D3.js.

There are 5 implemented visualizations:

1. Rene River

This is a line chart that allows to simultaneously view the variotions in tides of the Rene river through the last 5 years. Also, users can choose a specific year to view the information

2. Product Difference Chart

In this chart, users can select a specific time frame and a pair of products to be provided with a line chart of the difference in price between these to products accross the set time

3. Pie Chart

The purpose of this visualization is to allow analysts to easily understand how each product impacted the total expense of a certain time frame

4. Box Plot

With this box plot, users can extract a lot of valueble information about the price fluctuation of each product during the personalized time frame
